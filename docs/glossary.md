# Glosario del Proyecto

## Términos de Ingeniería de Datos

**Bronze/Raw**: Datos tal como llegan, sin modificar. Garantiza trazabilidad.

**Silver**: Datos limpios, tipados, con reglas de calidad aplicadas.

**Gold**: Datos modelados para consultas específicas (marts, métricas).

**DuckDB**: Motor de base de datos OLAP local, optimizado para análisis.

**Sanity Check**: Verificación básica de calidad (conteos, nulos, rangos).

**Hash SHA-256**: Huella digital del archivo para detectar cambios.

## Términos del Dominio (Exoplanetas)

**pl_name**: Nombre del planeta descubierto

**hostname**: Estrella anfitriona del planeta

**discoverymethod**: Método de descubrimiento (Transit, Radial Velocity, etc.)

**pl_orbper**: Periodo orbital en días

**pl_rade**: Radio del planeta en radios terrestres

**pl_bmasse**: Masa del planeta en masas terrestres

**sy_dist**: Distancia al sistema en parsecs

**st_teff**: Temperatura efectiva de la estrella (K)

**st_rad**: Radio de la estrella (radios solares)

**st_mass**: Masa de la estrella (masas solares)
